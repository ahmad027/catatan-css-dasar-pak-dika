# CSS Inheritance

Pengertiannya @ sebuah elemen mewarisi beberapa nilai dari properti yang dimiliki oleh elemen parent-nya 

Properti yang diwariskan

    color 
    font 
    letter-spacing 
    line-height 
    list-style 
    text-align 
    text-indent 
    text-transform
    visibility
    white-space 


elemen <a> secara default tidak mewarisi nilai dari properti color
